# help for git

# 下载

git clone 项目地址

# 提交

1. 添加至缓存区

  git add . 

  git add help.py 

2. 添加备注

  git commit -m '备注'

3. 提交至仓库

  git push origin main(仓库路径)

# 删除
## 删除某个文件
1. 删除文件 

  git rm help.py

2. 查看缓存

  git status

3. 提交

  git commit -m '备注'

  git push origin main
